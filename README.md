# Student Budget AI

### Система распределения бюджета студенческого самоуправления (CrewAI + Streamlit)

## Описание проекта

Данный проект реализует интеллектуальную систему распределения бюджета
студенческого самоуправления с использованием **AI‑агентов CrewAI** и
веб‑интерфейса **Streamlit**.

Система анализирует заявки студенческих клубов, учитывает стратегические
приоритеты университета и автоматически формирует обоснованное
распределение бюджета.

Проект разработан в рамках учебной работы:

**Тема 7 --- Распределение бюджета студенческого самоуправления**

------------------------------------------------------------------------

# Архитектура системы

Система использует **двух AI‑агентов**, которые работают
последовательно.

## Агент 1 --- Бюджетный аналитик

**Роль** Бюджетный аналитик студенческого самоуправления.

**Задача** - анализ заявок клубов - анализ приоритетов университета -
оценка ограничений бюджета - формирование чернового распределения
средств

**Функции** - анализ Excel‑данных - ранжирование клубов - оптимизация
расходов - формирование аналитического отчёта

------------------------------------------------------------------------

## Агент 2 --- Координатор студенческих инициатив

**Роль** Координатор студенческих инициатив.

**Задача** - анализ результатов первого агента - балансировка интересов
клубов - финальное распределение бюджета - формирование итогового отчёта

**Функции** - финальное решение по финансированию - объяснение логики
распределения - анализ рисков - рекомендации на будущее

# Как работает система

1.  Пользователь загружает Excel файл заявок клубов\
2.  Вводит стратегические приоритеты университета\
3.  Указывает общий бюджет\
4.  AI‑агенты анализируют данные и формируют распределение бюджета

------------------------------------------------------------------------

# Локальный запуск проекта

## Установка зависимостей

    pip install -r requirements.txt

## Настройка переменных окружения

Создать файл `.env`:

    MODEL=gemini/gemini-2.5-flash
    GOOGLE_API_KEY=YOUR_API_KEY
    GEMINI_API_KEY=YOUR_API_KEY

API ключ можно получить:\
https://ai.google.dev

------------------------------------------------------------------------

## Запуск демо режима

    python run_demo.py

Результат сохранится:

    reports/latest_budget_report.md

------------------------------------------------------------------------

## Запуск веб интерфейса

    streamlit run streamlit_app.py

После запуска откроется:

    http://localhost:8501
